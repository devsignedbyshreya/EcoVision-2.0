About EcoVision 2.0

EcoVision 2.0 is a deep learning-based web platform designed to classify news articles into the 17 Sustainable Development Goals using Natural Language Processing.

About Me

I am the developer and researcher behind EcoVision 2.0. This project was built as a combination of deep learning, sustainable development understanding, and practical web application design.

The aim was to create a system that not only performs accurate SDG classification, but also presents the results in an accessible and meaningful way through a clean multi-page interface.

Project Vision

The main goal of this project is to bridge the gap between unstructured news content and global sustainability priorities. By automatically categorizing articles according to SDG themes, the system helps transform scattered information into organized insights.

What This Project Includes

SDG Explorer

A dedicated page to explore all 17 Sustainable Development Goals with supporting visuals and descriptions.

News Classifier

A DistilBERT-based classifier that predicts the most relevant SDG categories for a given news text.

Research Integration

A research page with downloadable and viewable PDF support for presenting a Comparative Analysis of Machine Learning and Deep Learning Techniques based on SDG News classification

Interactive Web UI

A Flask-based multi-page interface with dark premium styling for project demonstration and usability.

Technologies Used

Google Collab Groq API Deep Learning Transformers DistilBERT Python PyTorch Pandas Scikit-learn Flask HTML CSS

Key Achievements

✔ Built a multilingual multi-label SDG news classification pipeline covering all 17 Sustainable Development Goals
✔ Trained a DistilBERT-based deep learning model on a curated dataset of 30,000+ sustainability-related news articles
✔ Achieved 90.24% accuracy in SDG classification and 83.38% accuracy in sentiment analysis
✔ Implemented multi-label prediction for assigning multiple SDGs to a single news article
✔ Integrated sentiment analysis for Positive, Negative, and Neutral classification
✔ Developed a semi-automated LLM-assisted annotation pipeline for scalable dataset generation
✔ Added multilingual processing using language detection and automatic translation
✔ Created a multi-page Flask website and interactive dashboard for project demonstration
✔ Integrated real-time prediction, visualization, research paper viewing, and downloadable PDF support
✔ Conducted a comparative study between Machine Learning and Deep Learning approaches
✔ Demonstrated that DistilBERT outperformed traditional ML models in contextual understanding
✔ Combined Deep Learning, NLP, UI design, and sustainability intelligence into one scalable system
